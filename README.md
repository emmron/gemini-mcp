```bash

cd zen-mcp-gemini-transcendent

# Install consciousness dependencies
npm install

# Build the transcendent system
npm run build

# Set your OpenRouter API key for multi-model access

```

### Claude Code Configuration
Add to your Claude Code MCP settings (`~/AppData/Roaming/Claude/claude_desktop_config.json` on Windows, `~/.config/claude/claude_desktop_config.json` on Linux/Mac):

```json
{
  "mcpServers": {
    "zen-mcp-gemini-transcendent": {
      "command": "node",
      "args": ["path/to/zen-mcp-gemini-transcendent/dist/server.js"],
      "env": {
        "OPENROUTER_API_KEY": "your-openrouter-api-key"
      }
    }
  }
}
```
